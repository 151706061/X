Method Detail
setColors(front, top, right, bottom, left, back)
Set side colors for this X.box. This triggers the re-creation of all vertices of this object after the colors are set. All colors must be given as 1-D arrays with the length 3 to hold R,G,B values.
Parameters:
{!Array} front
The X.color for the front (in positive Z direction) facing side.
{!Array} top
The X.color for the top (in positive Y direction) facing side.
{!Array} right
The X.color for the right (in positive X direction) facing side.
{!Array} bottom
The X.color for the bottom (in negative Y direction) facing side.
{!Array} left
The X.color for the left (in negative X direction) facing side.
{!Array} back
The X.color for the back (in negative Z direction) facing side.
Throws:
{X.exception}
If the given colors are invalid.
setSpacing(spacing)
Set the spacing for this box. By default, the spacing is 1.0. This triggers the re-creation of all vertices of this object after the spacing is set.
Parameters:
{number} spacing
The new spacing.
